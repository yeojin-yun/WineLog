# WineLog
와인 검색 및 추천에만 집중되어 있던 기존의 많은 앱들과는 다르게 나만의 와인 시음을 기록할 수 있는 앱.

## 기간
2022.10.03 ~ 2022.11.02

## 팀원
- 윤여진
- 이태형
- 서은지

## Requirements
- Language
    - Swift
- Framework
    - UIKit
    
## MVP
- 와인 추가
- 와인 정보 입력
- 와인 리스트

- MVC
